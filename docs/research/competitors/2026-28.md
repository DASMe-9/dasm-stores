## Delta جديد — W28

### 1. Salla: تفعيل ميزة الشراء السريع (Quick Buy)

| التفصيل | القيمة |
|---------|--------|
| المصدر | `changelog.salla.sa` — إدخال "تفعيل ميزة الشراء السريع" |
| الوصف | إضافة زر شراء مباشر من صفحة الكتالوج دون الانتقال لصفحة المنتج |
| النمط | يتحقق من وجود variants قبل إضافة للسلة؛ إن وُجدت: modal أو انتقال لتفصيل |
| الانطباق على dasm-stores | **نعم** — `components/product/ProductCard.tsx` لا يملك أي زر سلة حتى الآن |
| الملف المتأثر | `components/product/ProductCard.tsx` |

هذا ال Delta يُقدّم سياقاً تنافسياً مباشراً لـ spec هذه الجولة.

### 2. Shopify Dawn: Quick View على صفحات الكتالوج

| التفصيل | القيمة |
|---------|--------|
| المصدر | مجتمع مطوري Shopify 2026 (community.shopify.com) |
| الوصف | إضافة Quick View popup على بطاقة المنتج يُظهر صوراً + Add to Cart دون مغادرة الصفحة |
| الانتشار | واسع جداً في 2026 — معظم المتاجر الكبيرة تعتمده الآن |
| الانطباق على dasm-stores | **جزئياً** — المرحلة الأولى هي Quick Add مباشر (لا modal)؛ Quick View يأتي في مرحلة لاحقة |
| الملف المتأثر | `components/product/ProductCard.tsx` |

### 3. Shopify Dawn: Sticky Add-to-Cart للموبايل

| التفصيل | القيمة |
|---------|--------|
| المصدر | محادثات مجتمع Shopify 2026 |
| الوصف | شريط "إضافة للسلة" ثابت أسفل الشاشة عند التمرير في صفحات التفصيل |
| الانطباق على dasm-stores | **لا** — صفحة التفصيل (`app/[slug]/products/[productId]/page.tsx`) خارج نطاق هذه الجولة |
| الملف المتأثر | `app/[slug]/products/[productId]/page.tsx` |

### 4. Zid: Micro-interaction على زر السلة

| التفصيل | القيمة |
|---------|--------|
| المصدر | مصادر UX عامة (Trafiki, Baymard) + نمط Zid |
| الوصف | تأكيد بصري عند النقر: نبضة لون أو علامة ✓ خفيفة لمدة 200-300ms |
| الانطباق على dasm-stores | **نعم** — أي زر Quick Add مستقبلي يجب أن يتضمن هذا التأكيد |
| الملف المتأثر | `components/product/ProductCard.tsx` (مدمج في spec هذه الجولة) |
