## Delta جديد — W30

### 1. Shopify Dawn — Variant Swatches في بطاقات شبكة المنتجات (جديد)

| التفصيل | القيمة |
|---------|--------|
| المصدر | `github.com/Shopify/dawn` + `shopify.com/partners/blog/swatches` |
| الإصدار | Dawn 15.1+ (متاح فعلياً في 2026) |
| الوصف | Dawn أضافت دعماً أصلياً (native) لعرض خيارات اللون والحجم مباشرةً على بطاقة المنتج في صفحة الكتالوج — بدون الانتقال لصفحة التفصيل. الألوان تظهر كـ color circles، الأحجام كـ pills |
| النمط البصري | صف أفقي من circles/pills أسفل صورة المنتج، max 3-4 مرئية، "+ X" إن زادت |
| آلية Shopify | `swatch-input.liquid` + `swatch.liquid` + `swatch.color` Liquid object مع hex أو صورة |
| الانطباق على dasm-stores | **نعم** — `StoreProductCard` يحمل `variants?: StoreProductVariant[]` مع `option_values?: Record<string, string>`. البيانات متاحة. `ProductCard.tsx` حالياً لا يعرض أي variants |
| الملف المتأثر | `components/product/ProductCard.tsx` |
| Delta جديد؟ | **نعم** — لم يُوثَّق في تقارير W26-W29 |

### 2. Salla — تفعيل الشراء السريع (Instant Purchase)

| التفصيل | القيمة |
|---------|--------|
| المصدر | `help.salla.sa/article/تفعيل-الشراء-السريع` + `changelog.salla.sa/1511903425` |
| الوصف | ميزة "الشراء السريع" في Salla تتجاوز السلة كلياً — رابط أو زر يوجّه المشتري مباشرة لصفحة الدفع مع المنتج محدداً مسبقاً، بدون تسجيل دخول. مُصمّمة أصلاً لحملات الإعلانات المدفوعة |
| الانطباق على dasm-stores | **جزئياً** — مختلف عن Quick Add المُوثَّق في `product-card-quick-add-2026-06-13.md`. الشراء السريع يحتاج endpoint checkout خاصاً وقد يتجاوز نطاق الـ spec الواحد |
| الملف المتأثر | `app/[slug]/products/[productId]/page.tsx` + API جديد |
| Delta جديد؟ | **جزئياً** — الـ Quick Add الموثّق سابقاً هو "أضف للسلة بدون انتقال". Instant Purchase هي "تخطِّ السلة للدفع المباشر" — نمط أعمق. يُضاف لـ ideas-backlog |
