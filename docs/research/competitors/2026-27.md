## Delta جديد — W27

### 1. Salla: تفاصيل تنفيذ زر المفضلة (مصدر: SallaApp/theme-raed)

| التفصيل | القيمة |
|---------|--------|
| Selector | `.s-product-card-wishlist-btn[data-id=""]` |
| class عند الإضافة | `s-product-card-wishlist-added pulse-anime` |
| class عند الإزالة | `not-added un-favorited` |
| الموضع | **overlay على الصورة** في الكروت العمودية (vertical cards) |
| الموضع (horizontal/full-image) | في footer البطاقة مع زر السلة |
| تأثير بصري | `pulse-anime` — نبضة مرئية عند لحظة الإضافة |
| تحديث | Optimistic — يُبدّل الحالة فوراً قبل استجابة الـ API |

**الانطباق على dasm-stores:** هذا التفصيل يُكمل spec `product-tile-wishlist-2026-06-11.md` ويُغذّي spec هذه الجولة.

### 2. Shopify Dawn: اتجاه Quick-Add Overlay

من مجتمع مطوري Shopify (Dawn 11+):
- الاتجاه السائد: زر "Add to Cart" **صغير ودائري** (لا مستطيل) يظهر بـ **hover على صورة المنتج** لا أسفل البطاقة
- المطورون يحوّلون الزر من `btn--full` الثقيل إلى زر overlay دائري
- هذا النمط يطابق مباشرة "زر السلة الدائري" المذكور في baseline dasm-stores

**الانطباق:** يُعزز قرار جعل زر السلة في ProductCard دائرياً (`rounded-full`) وليس `rounded-xl` كما هو حالياً.

## فجوة جديدة — غير موثّقة في 2026-26

### ProductCard (store-level) — غياب المفضلة والسلة الدائرية

**الملف:** `components/product/ProductCard.tsx`

الـ baseline README يُصرّح: "صف ProductCard مع مفضلة" تحت قسم `subdomain-store`.

الحالة الراهنة في الكود:
```
store-product-card__body: name + price (فقط)
store-product-card__media: صورة + badge مميز + badge خصم (فقط)
لا زر مفضلة، لا زر سلة
```

| الغياب | المنافس المرجعي | الملف المتأثر | الحالة |
|--------|----------------|---------------|--------|
| زر مفضلة (قلب overlay) | Salla, Zid, Shopify | `components/product/ProductCard.tsx` | **محل spec هذه الجولة** |
| زر سلة دائري overlay | Shopify Dawn community | `components/product/ProductCard.tsx` | مرتبط بالـ spec |

**ملاحظة:** spec 2026-06-11 يغطّي ProductTile في `app/page.tsx` (marketplace). هذا الـ spec يغطي ProductCard في `components/product/` (store pages). مسار مختلف، مكوّن مختلف.
