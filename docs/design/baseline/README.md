## تحديث معتمد 2026-06-07 (موافقة محمد الزهراني)

أُعيد تجميد الـ baseline على تحسينات UX معتمدة من المالك. هذه هي **المرجع الجديد**،
وليست drift:

- **هيدر المتجر الفرعي:** Hero التركوازي مُصغّر (h-36 / md h-52) + رابط «← متاجر داسم»
  أصبح يعود للصفحة الرئيسية للسوق (`/`).
- **شبكة المنتجات:** 6 بطاقات في الصف على الويب (متجاوبة 2/3/4/6).
- **بطاقة المنتج:** صورة برسالة بديلة أنيقة عند الفشل (لا صورة مكسورة).
- **أيقونات الدفع:** mada / Visa / Mastercard / Apple Pay (متوافقة مع Paymob KSA).
- **لوحة تحكم البائع:** «أفضل المنتجات» تعرض حتى 12 منتجاً.

تقرير `docs/research/baseline-drift-2026-06-05.md` **متجاوَز** بهذا التحديث.

## الغرض

مرجع بصري ثابت لمنصة **متاجر داسم** (`dasm-stores`). أي مطابقة للتنفيذ مع هذا الـ baseline تتم عبر مقارنة مباشرة مع اللقطات والمكوّنات أدناه، وليس من الذاكرة أو من نسخ قديمة.

## مكوّنات مرئية حسب الشاشة

### marketplace-home

- هيدر المنصة (شعار، روابط، تسجيل دخول، سلة بعدد)
- **Hero (marketplace)** مع نص رئيسي/فرعي، بحث، وأيقونات مزايا
- صف **ProductCard** مع شارة رعاية، مفضلة، سعر ومتجر
- **AdSlot** مدمج في صف المنتجات + بانر عرض كامل
- **StoreCard** في صف متاجر
- شبكة **CategoryCard**
- **StatsBar** أسفل الصفحة

### subdomain-store

- هيدر المنصة (نفس منطق السوق)
- **Hero (store subdomain)** مع تعليق «منطقة فيديو دعائي متحرك»
- **StoreInfoCard** عائمة (شعار، اسم، وصف، وسوم موقع/ثقة/توصيل، أزرار سلة/تصفح/مشاركة)
- شريط بحث + قوائم «تبويب» و«ترتيب»
- ترويسة قسم «منتجات مميزة» وصف **ProductCard** مع مفضلة
